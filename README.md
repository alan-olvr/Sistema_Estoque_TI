# Sistema para Controle de Estoque: TI
Sistema para controle de estoque focado em empreendimentos de TI, em especifíco, lojas que trabalham com TI. O projeto contempla os problemas cotidianos desses empreendimentos, como ausência no controle de estoque, ausência no controle de ordens de serviço, ausência no controle de vendas etc.
